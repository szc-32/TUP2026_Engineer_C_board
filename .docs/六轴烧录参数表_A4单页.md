## 3) 首次联调步骤（简版）

1. 单轴联调，确认方向。
2. 方向反向则该轴 `direction = -1`。
3. 记录机械零位原始值，回填该轴 `zero_raw`。
4. 六轴确认完成后进入 `CALIBRATE`。
5. 校准成功后确认 `zero_offset` 已写入 flash。
