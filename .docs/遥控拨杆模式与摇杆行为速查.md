# 遥控拨杆模式与摇杆行为速查

> 适用分支：当前 `App/Src/system.cpp` 与 `App/Src/moving_behavior.cpp` 实现
>
> 通道定义：`ch0=右摇杆左右`，`ch1=右摇杆上下`，`ch2=左摇杆左右`，`ch3=左摇杆上下`，`ch4=拨轮`

## 1. 拨杆组合总表

| 左拨杆 | 右拨杆 | 模式(mode / engineer_mode) | 摇杆行为摘要 |
|---|---|---|---|
| 下 | 下 | `ZERO_FORCE_MOVE / ZERO_FORCE` | 无力保护，六轴不执行控制 |
| 下 | 中 | `RELATIVE_ANGLE / STOP_POSITION` | 常规底盘/云台控制链路 |
| 下 | 上 | `RELATIVE_ANGLE / USER` | 自定义控制器模式 |
| 中 | 下 | `RELATIVE_ANGLE / USER` | 自定义控制器模式 |
| 中 | 中 | `SPIN / KEYBOARD` | 小陀螺模式 |
| 中 | 上 | `NORMAL / INIT` | 六轴归位（Home流程） |
| 上 | 下 | `NORMAL / KEYBOARD` | 六轴末端平移专用（XYZ） |
| 上 | 中 | `NORMAL / KEYBOARD` | 六轴末端姿态/J4专用 |
| 上 | 上 | `NORMAL / KEYBOARD` | 键鼠/拨轮通用控制 |

## 2. 六轴专用摇杆映射（左上两档）

### 2.1 左上 / 右下：末端平移模式（XYZ）

- 左摇杆上下 `ch3` -> 末端 `Z`（上下）
- 左摇杆左右 `ch2` -> 末端 `Y`（左右）
- 右摇杆上下 `ch1` -> 末端 `X`（前后）

备注：此模式下会屏蔽键鼠/自定义控制器输入，避免叠加干扰。

### 2.2 左上 / 右中：末端姿态 + J4 模式

- 左摇杆上下 `ch3` -> 末端 `pitch`
- 左摇杆左右 `ch2` -> 末端 `roll`
- 右摇杆左右 `ch0` -> `J4` 关节（通过 `seed_q[3]` 增量）

备注：同样会屏蔽键鼠/自定义控制器输入，避免叠加干扰。

## 3. 关键说明

- 当前小陀螺的 `spin_flag` 自动切入逻辑已注释，但 `中/中` 拨杆直切小陀螺仍生效。
- 左上两档都是 `NORMAL / KEYBOARD`，由 `moving_behavior.cpp` 内部按拨杆状态再细分成两种六轴遥控子模式。
- 六轴遥控灵敏度可在 `App/Src/moving_behavior.cpp` 顶部宏中调整：
  - `DM_ARM_RC_X_GAIN / DM_ARM_RC_Y_GAIN / DM_ARM_RC_Z_GAIN`
  - `DM_ARM_RC_PITCH_GAIN / DM_ARM_RC_ROLL_GAIN / DM_ARM_RC_J4_GAIN`
  - `DM_ARM_RC_STICK_DEADZONE`

## 5. 校准手势速记（形象版）

### 5.1 手势图例（看摇杆方向）

- 上内八：左杆 `↗` + 右杆 `↖`
- 下内八：左杆 `↘` + 右杆 `↙`
- 上外八：左杆 `↖` + 右杆 `↗`
- 下外八：左杆 `↙` + 右杆 `↘`

记忆法：

- `内八` = 两根摇杆都指向中间
- `外八` = 两根摇杆都指向外侧
- `上/下` = 两根摇杆都朝上半区/下半区

### 5.2 当前版本校准触发手势

- 触发动作：`下内八 + 左右拨杆都在下档 + 持续约2秒`
- 形象理解：两根摇杆一起往中间下方夹紧不放

对应代码阈值（`App/Src/system.cpp`）：

- `ch0 < -600`
- `ch1 < -600`
- `ch2 > 600`
- `ch3 < -600`
- `s[LEFT]=DOWN` 且 `s[RIGHT]=DOWN`
- 持续计数 `calibrate_hold_need = 2000`

### 5.3 触发后流程（现场观察版）

1. 进入 `CALIBRATE` 工作流，普通拨杆模式暂不接管。
2. Stage1：云台回中（会看到初始化灯效/提示音）。
3. Stage2：机械臂零位偏置标定。
4. Stage3：陀螺仪标定。
5. 完成后自动切回 `KEYBOARD`。

备注：若只调底盘+云台，建议不要误触发上述手势，避免进入整套校准流程。
