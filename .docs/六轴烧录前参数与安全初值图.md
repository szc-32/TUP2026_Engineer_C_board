### 快速调参步骤

1. 单轴联调，确认方向是否正确。
2. 方向反向则该轴 `direction = -1`。
3. 记录机械零位原始值，回填该轴 `zero_raw`。
4. 六轴都确认后执行 `CALIBRATE`，写入 `zero_offset` 到 flash。
